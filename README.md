# Job Board Application 💼

A modern Android application built using Jetpack Compose to explore and manage job listings.

## Features ✨
* Dynamic list view of active job postings.
* Modern, responsive user interface with Jetpack Compose.
* Light and dark mode support out of the box.

## Built With 🛠️
* **Kotlin** - Language of choice for modern Android development.
* **Jetpack Compose** - Android’s modern toolkit for building native UI.
* **Material Design 3** - Latest design principles for clean user experience.
